# CBC Supplementary Material

Supplementary File 1. Supplementary table manifest. This file lists Supplementary Tables S1-S24, including table titles, repository paths, availability status, and archived-data-package coverage. The manifest points to the deposited frozen data package and repository files used to reproduce the manuscript analyses.

Supplementary File 2. Candidate flags and interpretation guardrails for the top 30. This editable TSV reports automatic audit flags, TME risk, normal-risk interpretation, maximum-risk organ, accessibility class, HPA status, discordance flags, missing primary components, isoform-resolution status, and candidate-specific caveats for the top 30 ranked candidates.
