# Limitations Register

This file records planned layers that are not included in the current MVP scoring pass. The purpose is to prevent optional annotations from silently becoming either blockers or untracked omissions.

| ID | Phase | Layer | Current decision | Reason | Effect on score | Planned handling |
|---|---|---|---|---|---|---|
| LIM-001 | Fase 8 | scRNA-derived `SC` score | Not included in MVP score | No processed annotated gastric scRNA dataset was admitted through the Fase 8 gate. Bulk TME and purity-adjusted correlations are available only as flags. | `SC=not_available`; no imputation. | Use TME flags in candidate cards and revisit cell-resolved validation if a suitable dataset is curated. |
| LIM-002 | Fase 10 | Universe-wide structural assessment | Deferred before Fase 13 | The plan defines structure as a candidate-card/tiering annotation, not a principal score component. Running AlphaFold/PDB over all 2,650 genes would add interpretation risk and low leverage before scoring stability is known. | No structural score enters the numeric ranking. | Run a light PDB/ECD/pLDDT annotation only for top preliminary candidates after Fase 13 if needed for candidate cards. |
| LIM-003 | Fase 11 | DepMap/survival functional evidence | Deferred before Fase 13 | Functional evidence is an incremental annotation and can be lineage/sample-count limited; it is not part of the preregistered principal score. | No functional score enters the numeric ranking. | Add as candidate-card/tiering evidence after preliminary ranking if gastric-lineage coverage is interpretable. |
| LIM-004 | Fase 12 | Full clinical/druggability automation | Deferred before Fase 13; candidate-level pass still expected | Clinical/druggability evidence is necessary for final candidate interpretation but is explicitly outside the numeric score. Full automation before a preliminary top set is low yield. | No clinical/druggability score enters the numeric ranking. | Curate at least top 30-50 preliminary candidates before final tiering and manuscript claims. |
| LIM-005 | Fase 4/13 | GPI-anchor source coverage | Partly resolved by Fase 4 correction | Confirmed UniProt lipidation `GPI-anchor` evidence is now counted universe-wide as strong anchor evidence; subcellular-location-only GPI annotations remain flags without direct credit. Some confirmed GPI genes still remain outside Core+Probable because other support criteria are not met. | GPI anchors are no longer systematically under-credited in `Surf`, but candidate cards still need manual review of shedding/soluble biology and localization context. | Carry GPI status into Fase 14 sensitivity and candidate cards; avoid treating GPI as experimental capture evidence. |

Current next step: Fase 14 sensitivity/stability on the active Fase 13 v2 preliminary ranking, with the failed aggregate positive-control top50 gate and passed cause-corrected gate both reported, and with explicit `SC=not_available`.
